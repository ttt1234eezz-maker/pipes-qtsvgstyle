# pipes-qt 

The Pipes theme for Qt.

## Included themes

- `theme/Pipes/pipes.qss` (dark for QtQSS)
- `theme/Pipes/pipes-light.qss`(light for QtQSS)

The theme provides a dark, cool-gray style for standard Qt widgets such as buttons, progress bars, menus, line edits, sliders, and selection controls.
